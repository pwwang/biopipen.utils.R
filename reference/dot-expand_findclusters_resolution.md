# Expand the resolution for `FindClusters`

So that we can have 0.1:0.4:0.1 to be expanded to 0.1, 0.2, 0.3, 0.4

## Usage

``` r
.expand_findclusters_resolution(resolution)
```

## Arguments

- resolution:

  A numeric vector or a character vector

## Value

A numeric vector with the expanded resolution
