# RunSeuratDEAnalysis

Run differential expression analysis on a Seurat object

## Usage

``` r
RunSeuratDEAnalysis(
  object,
  group_by,
  ident_1 = NULL,
  ident_2 = NULL,
  assay = NULL,
  subset = NULL,
  cache = NULL,
  error = TRUE,
  ...
)
```

## Arguments

- object:

  Seurat object

- group_by:

  Column name in meta.data to group cells by

- ident_1:

  Identity of the first group of cells

- ident_2:

  Identity of the second group of cells

- assay:

  Assay to use for analysis

- subset:

  Subset of cells to use for analysis It should be a string of
  expression to pass to
  [`dplyr::filter`](https://dplyr.tidyverse.org/reference/filter.html)
  function

- cache:

  Directory to cache the results. Set to `FALSE` to disable caching The
  results of the analysis will be saved in this directory Anything
  changed in the object or arguments will trigger a re-run

- error:

  Whether to raise an error if the analysis fails Otherwise, return an
  empty data frame

- ...:

  Additional arguments to pass to
  [`Seurat::FindMarkers()`](https://satijalab.org/seurat/reference/FindMarkers.html)
