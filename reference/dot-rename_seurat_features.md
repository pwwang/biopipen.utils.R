# Rename features in a Seurat object

Rename features in a Seurat object

## Usage

``` r
.rename_seurat_features(obj, features_map)
```

## Arguments

- obj:

  A Seurat object

- features_map:

  A named character vector, where the names are the original feature
  names and the values are the new names

## Value

A Seurat object with renamed features
